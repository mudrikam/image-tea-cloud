Panduan Instalasi

Sangat mudah:

Unduh:

git clone https://github.com/mudrikam/Image-Tea-mini.git
cd Image-Tea-mini

(Tidak tahu itu apa? Klik tombol hijau "Code" di halaman GitHub lalu "Download ZIP". Ekstrak ke folder manapun.)

Jalankan:

# Windows
Launcher.bat

(Klik dua kali file Launcher.bat. Hanya itu.)

# Linux & MacOS
chmod +x Launcher.sh
./Launcher.sh

Catatan: Saya menguji ini di VM karena belum mampu beli Mac. Tapi secara teori, menurut dokumentasi Python, ini harus berjalan di semua platform. Jika Anda menemukan cara yang lebih baik untuk menjalankan ini di Linux atau macOS, tolong buat pull request!

Selesai!
