Install Aplikasi

Download source code dari GitHub. Gak perlu install package manager ribet-ribet, tinggal download terus jalanin aja.

$ git clone https://github.com/mudrikam/Image-Tea-mini.git
$ cd Image-Tea-mini

Gak tau Git? Klik tombol hijau "Code" di halaman GitHub terus pilih "Download ZIP".

Windows
Launcher.bat

Double-click file Launcher.bat. Beres!

Linux & macOS
$ chmod +x Launcher.sh
$ ./Launcher.sh

Catatan: Gue test ini di VM soalnya belum sanggup beli Mac. Tapi secara teori harusnya jalan di semua platform. Kalau kamu nemuin cara yang lebih baik buat Linux/macOS, bikin pull request ya!
